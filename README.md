# Explore With Me

Платформа для поиска и организации событий, где пользователи могут создавать мероприятия, оставлять заявки на участие и формировать подборки. Проект реализован в микросервисной архитектуре с разделением на основной сервис управления событиями и сервис сбора статистики просмотров.

**Стек технологий:** Java 21, Spring Boot 3.3, Spring Data JPA, PostgreSQL 16, Docker, Maven, Lombok, JUnit, Mockito, Testcontainers

## Основные возможности

- **Управление событиями** — создание, редактирование, публикация и поиск мероприятий с фильтрацией по дате, категории, доступности мест
- **Система заявок** — подача, подтверждение и отклонение заявок на участие с учетом лимитов
- **Рейтинги** — возможность ставить лайки и дизлайки событиям
- **Подборки** — формирование тематических подборок событий с возможностью закрепления
- **Статистика** — сбор и агрегация данных о просмотрах с учетом уникальных посетителей

## Выводы

В результате разработки создано полностью рабочее микросервисное приложение с четким разделением ответственности между сервисами. Реализована гибкая система фильтрации и поиска событий, продумана логика управления состояниями (модерация событий, подтверждение заявок). Отдельный сервис статистики позволяет эффективно собирать и анализировать данные о просмотрах без нагрузки на основную БД. Проект покрыт тестами и готов к развертыванию в Docker-окружении.

## Структура сервисов

- ewm-service (основной API): http://localhost:8080
- stats-server (сервис статистики): http://localhost:9090
- ewm-db (PostgreSQL): localhost:5432
- stats-db (PostgreSQL): localhost:5433

## Развертывание и системные требования
